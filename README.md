### Book API - RESTful сервис для управления электронными книгами

Book API — это современное веб-приложение, разработанное на платформе Spring Boot, которое предоставляет полноценную систему для создания, хранения и управления электронной библиотекой. Сервис сочетает в себе мощный бэкенд с интуитивно понятным REST API, позволяя пользователям не просто хранить книги, а полноценно взаимодействовать с ними в цифровом формате. Ключевая ценность приложения заключается в его многофункциональности: от базовых операций с книгами до продвинутых возможностей, таких как автоматическая конвертация загруженных файлов в удобный для чтения HTML-формат и система отслеживания прогресса чтения для каждого пользователя. Это превращает обычное хранилище файлов в интерактивную читательскую платформу.

## Требования для запуска проекта

* Java Development Kit (JDK) 17 или новее
* Apache Maven 3.8+ или возможность использовать встроенный Maven Wrapper
* PostgreSQL 14+ (база данных)
* Git для управления версиями
* Windows 10/11 (64-bit)

## Основные функции:

* Аутентификация и авторизация
* Регистрация с подтверждением email
* JWT-аутентификация
* Две роли: пользователь и администратор

## Управление книгами

* Создание, редактирование, удаление книг
* Загрузка файлов книг (PDF, DOCX и др.)
* Автоматическая конвертация в HTML формат
* Поиск по автору и названию

## Чтение книг

* Просмотр книг в HTML формате
* Система отслеживания прогресса чтения
* Сохранение текущей страницы для каждого пользователя

## Технические особенности

* PostgreSQL для хранения данных
* Асинхронная обработка конвертации книг
* Swagger документация API
* Загрузка файлов на сервер
* Валидация входных данных

## Для кого предназначено:

* Для личного использования как цифровая библиотека
* Для образовательных учреждений
* Как основа для сервиса чтения книг онлайн

## Технологический стек:

* Backend: Spring Boot 3, Spring Security, JPA/Hibernate
* База данных: PostgreSQL
* Аутентификация: JWT
* Документация: OpenAPI/Swagger
* Сборка: Maven
